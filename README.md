# 📈 Crypto Vibe Check

A minimalist, high-performance market sentiment tracker built with **React** and **Framer Motion**. It fetches real-time data from the "Fear & Greed Index" to show whether the crypto market is panicking or feeling greedy.



## ⚡️ Key Features
* **Real-time Data:** Live fetching from Alternative.me API.
* **Smooth Motion:** Fluid animations for progress bars and card entries using `framer-motion`.
* **Dynamic UI:** The interface changes colors based on the current market "mood" (from Panic Red to Bullish Cyan).
* **Zero Bloat:** Written with clean, human-readable code. No over-engineered state management.

## 🛠 Tech Stack
* **Framework:** React (Vite)
* **Styling:** Pure CSS (Custom properties & Glassmorphism)
* **Animations:** Framer Motion
* **API Client:** Axios
